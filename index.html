URL Shortner API

Created by Daynil - Github Repo

Append a URL to the end of the base URL:

  • /new/https://www.google.com/

  • /new/http://www.freecodecamp.com/challenges/url-shortener-microservice

Output for a query:

{ "original_url" : 'http://www.freecodecamp.com/challenges/url-shortener-microservice', "short_url" : "https:/daynil-url-short.herokuapp.com/4" }
